# Dashboard Comunal de Cali - Comuna 17
## Qué incluye
- Mapa de cada barrio de la comuna (Piloto: Gran Limonar, Gran Limonar Cataya, La Hacienda)
- Cineclubes de Cali
- Elecciones Regionales Cali 2023
## Cómo iniciar
npm start
